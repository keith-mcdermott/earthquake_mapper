Earthquake Mapper

This interactive map lets you explore earthquake activity around the world using data from the USGS Earthquake Catalog API.

Users provide:

  • Year
  • Month (1-12)
  • Minimum Magnitude

Each earthquake is displayed as a point whose size and color reflect its magnitude. The map supports zooming and panning so you can examine seismic activity at global, regional, or local scales.

The application retrieves live data from the USGS in GeoJSON format and renders it dynamically.
